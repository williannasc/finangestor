# FinanGestor 💰

O **FinanGestor** é um sistema de controle financeiro pessoal desenvolvido para oferecer uma visão clara e objetiva do fluxo de caixa mensal e acumulado. O projeto foca em **Glassmorphism Design**, usabilidade rápida e precisão matemática no saldo livre.

---

## 🚀 Funcionalidades Principais

- **Dashboard Inteligente**: Visualização de 4 KPIs essenciais (Receita Mensal, Cartão de Crédito, Despesas Fixas e Despesas Totais).
- **Cálculo de Saldo Acumulado**: Diferente de apps comuns, o sistema calcula o saldo livre baseando-se no que sobrou dos meses anteriores, somando a receita atual e subtraindo os gastos.
- **Lançamento Rápido**: Interface otimizada para lançar despesas em segundos diretamente na home.
- **Gestão de Cartões**: Monitoramento de limites e gastos específicos por bandeira/banco.
- **Extrato Detalhado**: Listagem com identificação visual de método de pagamento (Cartão ou Dinheiro/Pix) e controle de parcelamento (ex: 1/12).
- **Navegação Temporal**: Filtro dinâmico para navegar entre meses e anos.

## 🛠️ Tecnologias Utilizadas

- **Backend**: PHP 8.x
- **Banco de Dados**: MySQL / MariaDB
- **Frontend**: Tailwind CSS (via CDN)
- **Ícones**: Google Material Icons
- **Design**: Custom Glassmorphism CSS

## 📋 Pré-requisitos

- Servidor Web (Apache/Nginx)
- PHP >= 7.4
- MySQL
